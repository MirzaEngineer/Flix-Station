# 🎬 CineStream-X (Full-Stack MERN Netflix Clone)

A feature-rich, full-stack movie streaming platform that replicates the Netflix experience. [cite_start]This project demonstrates end-to-end development, from managing a **MongoDB** database to building a responsive **React** UI. 

---

## 🚀 Key Features

* **User Authentication**: Secure Login/Signup using **JWT (JSON Web Tokens)** and **Bcrypt**.
* **Personalized Watchlist**: Users can save movies to their profile, with data persisted in **MongoDB**.
* **Real-time API Integration**: Fetches the latest movie data and trailers from the **TMDB API**.
* **Responsive Video Player**: Seamless trailer playback using `react-youtube`.
* [cite_start]**State Management**: Efficient data flow handled via **Redux** or **Context API**.
